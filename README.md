## 它是如何工作的？

XArchiver 是一款基于大数据的压缩包密码破解软件，它并不穷举所有可能性，而仅遍历那些网上经常出现的解压密码，在大幅提升破解效率的同时成功率也远高于传统的暴力破解方式。

基于以上原理，XArchiver 仅对那些你从网络下载却不知道出处的压缩包资源有奇效。如果你已明确知晓其来源，或者压缩包本身就是你创建的只不过忘记了密码，那么软件能起到的作用将非常小。

> [!TIP]
> XArchiver 收集了 10w+ 中文互联网常见的解压密码，破解可能需要较长时间，期间请勿运行其它高资源占用的程序并耐心等待。
